## ما هذا؟

نظام تشغيل مجاني للمحامين المستقلين في دبي. يقوم بفهرسة ملفات القضايا وملاحظات البحث ومراسلات العملاء ويجعلها قابلة للبحث الفوري. جميع بياناتك تبقى على جهازك — لا سحابة، لا اشتراك.

**مدينة واحدة، نظامان** — المشهد القانوني الفريد في دبي: مركز دبي المالي العالمي (القانون العام · الإنجليزية) + البر الرئيسي (القانون المدني · العربية) — في أداة واحدة.

اختر نظامك عند التثبيت: `--system difc` أو `--system mainland` أو `--system both`.

## التثبيت

```bash
cd ~/ai-law-firm-dubai-difc
pip install -e .
```

## البداية السريعة

```bash
# 1. إعداد مكتبك (اختر نظامك)
ailawfirm-dubai init ~/matters --system mainland    # القانون المدني للبر الرئيسي
ailawfirm-dubai init ~/matters --system difc       # القانون العام لمركز دبي المالي
ailawfirm-dubai init ~/matters --system both        # النظام المزدوج

# 2. فهرسة ملفاتك
ailawfirm-dubai mine ~/matters

# 3. البحث في كل شيء
ailawfirm-dubai search "عقد تجاري"
ailawfirm-dubai search "goAML"
```

## تبديل النظام

يمكنك التبديل بين نظامي DIFC والبر الرئيسي في منتصف الجلسة:

```
"switch to DIFC"     → سياق DIFC (القانون العام · الإنجليزية)
"switch to Mainland" → سياق البر الرئيسي (القانون المدني · العربية)
```

## المحاكم المدعومة (8 محاكم)

**محاكم DIFC:** المحكمة الابتدائية · محكمة الاستئناف · محكمة التمييز · محكمة الدعاوى الصغيرة

**محاكم البر الرئيسي:** محكمة دبي الابتدائية · محكمة الاستئناف · محكمة التمييز · مركز فض المنازعات الإيجارية

## الامتثال التنظيمي

يكتشف وكيل الامتثال الكلمات المفتاحية لـ:
- **goAML** — مكافحة غسل الأموال إلزامي
- **DPL** — قانون حماية البيانات (رقم 5 لسنة 2020)
- **UAE DPP** — المرسوم الاتحادي رقم 45 لسنة 2021
- **CLPD** — 16 نقطة/سنة إلزامية لتجديد الترخيص
- **قيود الدعاية والإعلان** — قانون مهنة المحاماة
