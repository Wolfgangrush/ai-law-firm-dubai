## System
## System: DIFC

## Authority Tier
Court Rules (RDC Part 13)

## Language Variants
English (Primary)

## Currency Warning
DIFC Default Judgment available under RDC Part 13 where defendant fails to (a) file Acknowledgment of Service within prescribed period (typically 14 days · 28 days for out-of-jurisdiction service); or (b) file Defence within prescribed period. Available administratively for liquidated debt claims OR by interlocutory application for unliquidated claims (assessment hearing).

## Template Type
Court Form / Request

## Structural Skeleton

**(A) LIQUIDATED CLAIM — REQUEST FOR DEFAULT JUDGMENT**

1. **Heading** — DIFC Courts · case number · parties.
2. **Title** — "Request for Default Judgment".
3. **Statement of default**:
   - Date Claim Form served on defendant.
   - Whether Acknowledgment of Service filed (date if yes; "none filed" if no).
   - Whether Defence filed (date if yes; "none filed" if no).
   - Period elapsed beyond deadline.
4. **Sum claimed**:
   - Principal.
   - Interest (DIFC Court rate · contractual rate).
   - Costs (fixed costs).
5. **Annexures** — Certificate of Service · partial-payment evidence (if any).
6. **Signature** — solicitor.

**(B) UNLIQUIDATED CLAIM — APPLICATION**

1. **Application Notice** with supporting witness statement.
2. **Reliefs sought**:
   - Default judgment.
   - Assessment of damages by Registrar / Judge.
   - Costs.
3. **Hearing date** — assessment of damages.

## Drafting Conventions
- **Certificate of Service** — strict compliance; defective service nullifies default judgment.
- **Time-limit calculations** — calendar days; date of service vs date of issue.
- **Liquidated vs unliquidated** — different procedural paths.
- **Interest computation** — pre-judgment under DIFC Court Rate (typically EIBOR + margin) or contractual.
- **Fixed costs** — RDC fixed-fee scales for liquidated claims.
- **Service overseas** — different time-limits per RDC Part 9 + Hague Service Convention (UAE acceded).
- **Setting-aside grounds** — see file 79.
- **Foreign-jurisdiction debt** — DIFC Court accepts foreign-currency claims; converted at judgment date for execution.

## Fair-Use Excerpt
"A claimant may obtain default judgment against the defendant if the defendant has not filed an Acknowledgment of Service or Defence within the period required by these Rules." (RDC Part 13, paraphrased.)

## Practical Use for Solo Advocate / Small Firm
Default judgment is the principal commercial recovery tool in DIFC for clean liquidated claims. Solo practitioners must keep meticulous service records — service-step failures nullify the judgment. Default judgment is enforceable like any other DIFC judgment (cross-reference enforcement files 80-85).
