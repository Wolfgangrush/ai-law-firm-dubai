## یہ کیا ہے؟

دبئی کے سولو وکلا کے لیے ایک مفت، لوکل فرسٹ پریکٹس OS۔ یہ آپ کی کیس فائلوں، تحقیقی نوٹس، اور کلائنٹ کمیونیکیشنز کو انڈیکس کرتا ہے اور انہیں فوری طور پر قابل تلاش بناتا ہے۔ آپ کا ڈیٹا آپ کی مشین پر رہتا ہے — کوئی کلاؤڈ نہیں، کوئی سبسکرپشن نہیں۔

**ایک شہر، دو نظام** — دبئی کا منفرد قانونی منظرنامہ: DIFC آف شور (کامن لا · انگریزی) + مین لینڈ (سول لا · عربی) — ایک ٹول میں۔

## انسٹالیشن

```bash
cd internal-paths/ai-law-firm/dubai-difc
pip install -e .
```

## فوری آغاز

```bash
ailawfirm-dubai init ~/matters --system both
ailawfirm-dubai mine ~/matters
ailawfirm-dubai search "آپ کی تلاش"
```
