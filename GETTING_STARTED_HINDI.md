# शुरुआत कैसे करें — AI Law Firm · दुबई-DIFC · v0.1

```
🙏 स्वागत है, वकील साहब — आपकी AI Law Firm यहाँ से शुरू होती है।
```

## यह क्या है?

दुबई के सोलो वकीलों के लिए एक मुफ्त, लोकल-फर्स्ट प्रैक्टिस OS। यह आपकी केस फाइलों, रिसर्च नोट्स और क्लाइंट कम्युनिकेशन को इंडेक्स करता है और उन्हें तुरंत खोजने योग्य बनाता है। आपका डेटा आपकी मशीन पर रहता है — कोई क्लाउड नहीं, कोई सब्सक्रिप्शन नहीं।

**एक शहर, दो सिस्टम** — दुबई का अनूठा कानूनी ढांचा: DIFC ऑफशोर (कॉमन लॉ · अंग्रेजी) + मेनलैंड (सिविल लॉ · अरबी) — एक टूल में।

## इंस्टॉलेशन

```bash
cd internal-paths/ai-law-firm/dubai-difc
pip install -e .
```

## तुरंत शुरुआत

```bash
ailawfirm-dubai init ~/matters --system both
ailawfirm-dubai mine ~/matters
ailawfirm-dubai search "आपकी खोज"
```

## समर्थित अदालतें (8 अदालतें)

**DIFC अदालतें:** CFI · CA · Cassation · SCT
**मेनलैंड अदालतें:** दुबई CFI · CA · Cassation · RDC (किराया विवाद केंद्र)

## अनुपालन (Compliance)

compliance_agent चेतावनी देता है: goAML (एंटी मनी लॉन्ड्रिंग) · DPL (डेटा प्रोटेक्शन) · CLPD (लाइसेंस नवीनीकरण) · प्रचार प्रतिबंध।
