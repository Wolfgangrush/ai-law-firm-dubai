# PAGSISIMULA — AI Law Firm · Dubai-DIFC · v0.1

```
🙏 Mabuhay, abogado — magsisimula ang iyong AI Law Firm dito.
```

## Ano ito?

Isang libre, local-first na practice OS para sa mga solo advocate sa Dubai. Ini-index nito ang iyong mga case file, research notes, at client communications — at ginagawang agad na mahanap ang mga ito. Ang iyong data ay nananatili sa iyong machine — walang cloud, walang subscription.

**Isang Lungsod, Dalawang Sistema** — ang natatanging legal na tanawin ng Dubai: DIFC offshore (common-law · English) + Mainland (civil-law · Arabic) — sa isang tool.

## Instalasyon

```bash
cd ~/ai-law-firm-dubai-difc
pip install -e .
```

## Mabilis na Pagsisimula

```bash
ailawfirm-dubai init ~/matters --system both
ailawfirm-dubai mine ~/matters
ailawfirm-dubai search "iyong paghahanap"
```

## Mga Suportadong Hukuman (8 hukuman)

**DIFC Courts:** CFI · CA · Cassation · SCT
**Mainland Courts:** Dubai CFI · CA · Cassation · RDC (Rental Disputes Center)

## Pagsunod (Compliance)

Ang compliance_agent ay nagbabala: goAML (anti-money laundering) · DPL (data protection) · CLPD (lisensya) · mga paghihigpit sa publisidad.
